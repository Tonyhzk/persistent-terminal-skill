# 更新日志

## [1.2.0] - 2026-02-12

### 新增
- **read 输出到文件** - `--output` 参数将输出写入文件而非 stdout，节省对话上下文
- **read 字符截断** - `--max-chars` 参数限制最大输出字符数（默认 2000，0 为不限制）

### 改进
- **read 默认行数** - 从 50 行降为 30 行，减少上下文占用
- **exec 超时行为** - 超时不再返回历史面板内容，仅提示用 read 查看，避免无关输出污染上下文
- **路径定位简化** - 日志和会话目录改用 `cwd` 相对路径，不再依赖 `.claude` 目录查找

---

## [1.1.0] - 2026-02-11

### 新增
- **send 命令** - 支持发送纯文本到会话，适用于密码等交互式输入
- **JSON 配置读取** - send 命令支持 `--config` + `--key` 从 JSON 文件读取文本，绕过 bash 特殊字符问题
- **后台模式** - create 命令支持 `--background` 参数，静默创建会话不弹出窗口

### 改进
- **会话管理** - 优化会话状态持久化，使用文件系统存储会话信息
- **跨平台兼容** - 完善 Windows 子进程执行模式

---

## [1.0.0] - 2026-02-11

### 首次发布

- **持久终端会话** - 跨多轮 Claude Code 对话保持同一终端会话
- **跨平台支持** - macOS/Linux 使用 tmux，Windows 使用子进程模式
- **会话操作** - 支持 create、attach、exec、read、list、close、close-all 命令
- **前台附着** - 默认弹出系统终端窗口，实时查看输出
- **JSON 输出** - 所有命令返回结构化 JSON 格式
- **符号链接工具** - 提供 setup_claude_dir.py 实现多项目共享配置
